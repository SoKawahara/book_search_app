# Boopple

## サービス名の由来

**Boopple** は **Book（本）** と **Ripple（波紋）** を掛け合わせた造語です。  
本を通して人々のつながりが波紋のように広がっていってほしい、という願いが込められています。

---

## サービス概要

Boopple は、読書が苦手な人・これから本を読み始めたい人に向けた、本の感想投稿 SNS です。

「不特定多数のおすすめ本リスト」ではなく、**自分と境遇・価値観が近いユーザーのおすすめ本**を見つけることに特化しています。  
人間的に共感できる相手が紹介する本なら読んでみたいと思える——そんな体験を提供します。

---

## 背景・課題

読書習慣のない人が本屋に行っても、大量の本の中から「自分に合った一冊」を見つけるのは難しいです。  
「おすすめ本 ○○ 選！」のような不特定多数向けのリストも、自分にフィットするとは限りません。

一方で、**自分が好きな芸能人やインフルエンサーが紹介している本**なら読んでみようと思えた経験はないでしょうか。  
これは「本そのものへの興味」よりも「その人への共感・信頼」が読書のきっかけになっているということです。

Boopple はこの気づきをベースに、**自分と似た読書体験・価値観を持つユーザーを見つけ、その人がおすすめする本に出会える**仕組みを提供します。

---

## 対象ユーザー

- 読書を始めてみたいが、何から読めばよいかわからない人
- 本を読んだことはあるが、自分に合った本をなかなか見つけられない人
- 他者の読書体験から共感できる本を探したい人

---

## 主な機能

| 機能 | 概要 |
|------|------|
| **エピソード閲覧** | ユーザーが登録した「読書を始めたきっかけ」エピソードを閲覧できる。年齢・職業・エピソード内容などで絞り込み検索が可能。共感できる人のプロフィールへ遷移しておすすめ本を確認できる。 |
| **本の感想投稿** | 読んだ本の感想を投稿できる。読みやすさ・利便性・おすすめ度などを評価形式で記録。 |
| **いいね / フィード** | 投稿へのいいねと、フォロー中ユーザーの投稿フィード表示。いいねした投稿の一覧管理も可能。 |
| **マイ本棚** | 気になった本を「未読」「読了」の 2 状態で管理できる本棚機能。 |
| **本の検索** | 書名・著者名などで本を検索（Google Books API 連携）。縦一覧・タイル式の表示切替、並び替えに対応。 |
| **フォロー / フォロワー** | ユーザー間のフォロー機能。フォローしているユーザーの投稿をフィードで確認できる。 |
| **プロフィール設定** | 年齢・職業・読書を始めたきっかけとなるエピソードなどのプロフィールを設定できる。 |
| **アカウント認証** | メールによるアカウント有効化、パスワードリセット機能。 |

| Ruby on Rails 7.2.1 | Web アプリケーションフレームワーク |

| SQLite3 | 開発環境データベース |

| Kaminari | ページネーション |

| Bootstrap (Sass) | UI コンポーネント・レイアウト |
